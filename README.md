# Offline-First React Native Notes App

This is a clean, considered, offline-first notes application built with React Native and Expo. It is specifically designed to allow users to manage their notes without an internet connection, ensuring local persistence across app restarts.

## Features

- **Notes List:** View all your saved notes with a clear title and a short preview of the content.
- **Create / Edit Note:** A clean form with title and body fields to compose or update your notes.
- **Delete Note:** Easily remove notes you no longer need via a simple delete button on the list item.
- **Offline Persistence:** All notes are stored locally on the device. Close the app and reopen it, and your notes will still be there. No backend, no cloud sync.
- **Clean UI:** Considered typography, consistent spacing, and logical visual hierarchy that provides a polished, client-ready feel without unnecessary complexities.

### How to Install and Run (< 5 Minutes)

1. **Download the APK:** Tap the Google Drive link above on your Android device and download the `.apk` file.
2. **Allow Installation from Unknown Sources:**
   - When you try to open the downloaded APK, Android may prompt you that the app is from an unknown source.
   - Tap "Settings" in the prompt and toggle the switch to "Allow from this source" for your browser or file manager.
3. **Install the App:** Go back to the downloaded file, tap it again, and select "Install".
4. **Open the App:** Once installed, tap "Open" to launch the Offline Notes app immediately and start creating your notes!
